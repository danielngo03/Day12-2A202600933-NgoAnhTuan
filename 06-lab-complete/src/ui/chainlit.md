# 🔬 Lab10 – RAG Research Assistant

Welcome to the **Lab10 RAG Research Assistant** — an intelligent Q&A system built on top of a scholarly paper corpus fetched from the [Crossref API](https://www.crossref.org/).

## 🚀 What this assistant can do

- **Answer questions** about research papers on *agentic retrieval augmented generation*
- **Cite sources** directly from the indexed corpus with DOI links
- **Switch LLM models** between Gemini Flash Lite, OpenAI o4-mini, and Nemotron (OpenRouter)
- **Switch embedding models** between local MiniLM and cloud-based Nemotron Embed

## 💡 Example questions

- *"What is the main contribution of the paper on RAG agents?"*
- *"Who authored papers about retrieval-augmented generation?"*
- *"Which papers were published after 2024?"*
- *"Summarize the latest research on agentic LLM systems."*

## ⚙️ Settings

Use the settings panel (top-right ⚙️ icon) to:
- Select LLM model
- Select embedding model
- Adjust Top-K retrieval count
